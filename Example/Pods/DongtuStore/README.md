# dongtustore-ios-pod
 是上海动宇科技有线公司（上海似颜绘科技有限公司）提供的表情云商店SDK。开发者可以利用本SDK集成丰富的表情。https://biaoqingsdk.com

## Requirements
ios 8.0

## Usage
DongtuStore is available throuth [Cocoapods](http://cocoapods.org).To install it, simply add the following line to your Podfile.

```ruby
pod "DongtuStore"
```
