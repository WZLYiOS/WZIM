## Requirements:
- **iOS** 8.0+
- Xcode 10.0+
- Swift 5.0+


## Installation Cocoapods
<pre><code class="ruby language-ruby">pod 'WZEncryption', '~> 1.0.0'</code></pre>

## 导入头文件
```Objective-C
#import <WZDES3.h>
#import <WZMBase64.h>
```
